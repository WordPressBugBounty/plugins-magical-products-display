== Installation ==

= Automatic Installation (Recommended) =

1. Log in to your WordPress admin dashboard
2. Navigate to **Plugins > Add New**
3. Search for "Magical Shop Builder"
4. Click **Install Now** and then **Activate**
5. Go to **Magical Shop Builder** menu to configure settings
6. Start building with Elementor!

= Manual Installation =

1. Download the plugin zip file
2. Log in to your WordPress admin dashboard
3. Navigate to **Plugins > Add New > Upload Plugin**
4. Choose the downloaded zip file and click **Install Now**
5. Activate the plugin
6. Go to **Magical Shop Builder** menu to configure

= FTP Installation =

1. Download and unzip the plugin file
2. Upload the `magical-products-display` folder to `/wp-content/plugins/`
3. Activate the plugin through the **Plugins** menu in WordPress
4. Configure settings under **Magical Shop Builder** menu

= After Installation =

1. Ensure WooCommerce and Elementor are installed and activated
2. Visit **Magical Shop Builder > Dashboard** for quick start guide
3. Enable/disable widgets under **Magical Shop Builder > Widgets**
4. Create custom templates under **Magical Shop Builder > Templates**
5. Start editing pages with Elementor and use the new widgets!

== Frequently Asked Questions ==

= What is Magical Shop Builder? =

Magical Shop Builder (formerly Magical Products Display) is a complete WooCommerce template builder for Elementor. It provides 60+ widgets to build custom single product pages, cart pages, checkout flows, my account pages, shop archives, and more - all without coding.

= Is this plugin free? =

Yes! The core plugin with 60+ widgets and the template builder system is completely free. A Pro version is available with additional features like product comparison tables, countdown timers, hotspots, and more.

= Do I need Elementor Pro? =

No! Magical Shop Builder works perfectly with the FREE version of Elementor. You get template builder functionality similar to Elementor Pro's Theme Builder, but at no cost.

= Do I need WooCommerce? =

Yes, WooCommerce must be installed and activated. This plugin extends WooCommerce functionality with custom Elementor widgets.

= What happened to Magical Products Display? =

Magical Products Display has been completely rebuilt and renamed to Magical Shop Builder. All your existing widgets still work - we've just added 50+ more widgets and a complete template builder system. Updates are seamless with no migration needed.

= Can I build custom single product pages? =

Absolutely! Use our 16 single product widgets including Product Title, Price, Gallery, Add to Cart, Reviews, Tabs, Related Products, and more to design unique product page layouts.

= Can I customize the checkout page? =

Yes! We provide 10 checkout widgets including Billing Form, Shipping Form, Order Review, Payment Methods, and even a Multi-Step Checkout widget for step-by-step checkout flows.

= Will this slow down my site? =

No. We use on-demand asset loading - CSS and JavaScript files only load when you actually use the widgets. The plugin is optimized for performance with efficient database queries.

= Is it compatible with my theme? =

Magical Shop Builder works with any properly coded WordPress theme. The template builder lets you override theme templates for complete design freedom.

= Can I use this with other page builders? =

No, this plugin is specifically designed for Elementor page builder. You need Elementor (free or pro) installed and activated.

= How do I create a custom product page template? =

1. Go to **Magical Shop Builder > Templates**
2. Click **Add New Template**
3. Select "Single Product" as the template type
4. Design your template using single product widgets
5. Set display conditions (all products, specific categories, etc.)
6. Publish and it will automatically apply to your products

= Is the plugin translation ready? =

Yes! The plugin is fully internationalized and available in 10 languages. You can also contribute translations or create your own.

= Where can I get support? =

Visit our [support page](https://wpthemespace.com/contact/) or the WordPress.org support forum. Pro users get priority email support.

= How do I report a bug or request a feature? =

Please use the WordPress.org support forum or contact us directly at [wpthemespace.com/contact](https://wpthemespace.com/contact/).
